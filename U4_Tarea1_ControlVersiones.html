Qué es Gil y Github

Git es un sistema de control de versiones desarrollado por Linux Tarvalds (el hombre que creó Linux). Su considera por lo tanto que Gil es el corazón de Fithub. Si GIl es el corazón de GitHub, entonce HUb es su alma. El hub de Github es lo que convierte una linea de comandos, como Git, en la red social más grande para desarrolladores.

¿Qué es un sistema de control de versiones?

Cuando los desarrolladores hacen un nuevo proyecto, siempre continúan haciéndole modificaciones el código. Incluso después de la puesto en marcha de los proyectos, todavía necesitan actualizar las versiones, corregir errores, agregar nuevas funciones, etc.

El sistema de control de versiones ayuda a registrar el c'odigo borrado o modifucado.
